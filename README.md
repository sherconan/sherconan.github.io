# sherconan.github.io

Personal portfolio site — [sherconan.github.io](https://sherconan.github.io)

Dark-themed, responsive single-page portfolio showcasing AI + Finance projects. Built with vanilla HTML/CSS/JS, no frameworks.

## Features

- Starfield background animation
- Category filter (Finance, Knowledge, Tools, Scraper, Demo)
- Featured project badges
- Tech stack pills
- Responsive grid layout
